위드 임베딩: 단어를 벡터로 표현하는 것

단어 -> 밀집 표현

희소 표현, 밀집 표현, 워드 임베딩

# 1. 희소 표현(Sparese Representation)
앞서 원-핫 벡터처럼 값 대부분이 0으로 표현되는 방법을 희소 표현이라고 함.
원-핫 벡터는 희소 벡터임

문제점: 단어 개수가 늘수록 벡터의 차원이 한없이 커진다., 공간적 낭비

벡터 간 유사도 표현할 수 없음

# 2. 밀집 표현
밀집 표현: 벡터의 차원을 단어 집합의 크기로 상정하지 않음. 사용자가 설정한 값으로 모든 단어의 벡터 표현의 차원을 맞춘다. 0과 1이 아닌 실수값을 가지게 된다.

밀집 벡터

# 3. 워드 임베딩
단어를 밀집 벡터의 형태로 표현하는 방법을 워드 임베딩이라고 한다.
나온 결과를 임베딩 벡터라고 한다

nn.embedding() : 단어를 랜덤한 값을 가지는 밀집 벡터로 변환한 뒤에, 인공 신경망의 가중치를 학습하는 것과 같은 방식으로 단어 벡터를 학습하는 방법을 사용.

| 헤더1 | 원-핫 벡터 | 임베딩 벡터 |
| --- | --- | --- |
| 차원 | 고차원(단어 집합의 크기) | 저차원 |
| 다른 표현 | 희소 벡터의 일종 | 밀집 벡터의 일종 |
| 표현 방법 | 수동 | 훈련 데이터로부터 학습함 |
| 값의 타입 | 0, 1 | 실수 |
